## 🚀 Quick Start

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/devassist-bot.git
   cd devassist-bot
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Configure environment:
   ```bash
   cp .env.example .env
   # Edit .env with your tokens
   ```

4. Run tests:
   ```bash
   pytest tests/
   ```

5. Start the bot:
   ```bash
   python src/main.py
   ```
